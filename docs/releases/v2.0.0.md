## ‚ú® New Features & Functions

### Core Architecture
- **Orchestrator (`vauban.py`)**: A new Python-based central command system that manages the entire lifecycle of a scan (Recon -> Discovery -> Scanning -> Reporting).
- **Modular Design**: Functional separation into `recon`, `urls`, `api`, `scan`, and `report` modules for easier maintenance and upgrades.
- **Multithreading**: Enhanced performance with threaded execution for parallel tasks.

### Modules Added
#### 1. Reconnaissance
- **`subdomain.sh`**: Integrated `subfinder`, `assetfinder`, `amass`, and `crt.sh` for deep subdomain enumeration.
- **`dns.sh`**: Added `dnsx` for high-speed DNS resolution and wildcard filtering.
- **`techdetect.py`**: New Python module using `httpx` and custom signatures to fingerprint technologies and WAFs.

#### 2. URL Discovery
- **`passive.sh`**: Collects historical URLs from Wayback Machine, AlienVault, and CommonCrawl.
- **`crawler.sh`**: Active crawler integration using `katana` and `hakrawler`.
- **`jsparser.py`**: Dedicated JavaScript analyzer to extract endpoints and hardcoded secrets from JS files.

#### 3. API Discovery
- **`endpoints.sh`**: focused API endpoint discovery using `kiterunner`.
- **`openapi.py`**: Automatically detects and parses Swagger/OpenAPI documentation.
- **`params.sh`**: Fuzzes for hidden parameters using `arjun`.

#### 4. Vulnerability Scanning
- **`nuclei.sh`**: Orchestrates `nuclei` scans with customizable templates and severity filters.
- **`secrets.py`**: A new regex-based engine detecting 20+ types of secrets (AWS, Stripe, Github tokens, etc.).
- **`custom.py`**: Specialized checks for logic bugs like IDOR, CORS misconfigurations, and Race Conditions.

#### 5. Reporting & Notification
- **`generator.py`**: Generates a beautiful, Vauban-themed HTML report dashboard.
- **`notifier.py`**: Multi-channel notification support (Slack, Discord, Telegram).
- **`logger.py`**: Custom Rich-library based logger with "Vauban" branding and progress tracking.

### User Experience
- **`--check` Command**: Verifies installation status of all dependencies.
- **Installation Script**: Automated setup for all Go and Python dependencies.
- **Configuration**: Centralized `settings.yaml` for easy tuning of threads and timeouts.
